# create-template-cli

## Run tests
>`$ python -m unittest discover -s test/`

## Use

### help
> `$ python app/main.py -h || --help`

### API Type
> `$ python app/main.py   --type {fastapi,express,next}, -t {fastapi,express,next}` 

### Output directory
>`$ python app/main.py   --output OUTPUT, -o OUTPUT`
